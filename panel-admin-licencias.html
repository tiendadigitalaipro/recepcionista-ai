📖 Guía de Operación Completa

Cómo funciona todo el sistema paso a paso

PASO 1 · Encontrar Clientes
1. Abrir finder.py en la terminal: python finder.py --tipo "nail salon" --ciudad "Miami, FL" --max 50
2. El script busca negocios en Google Maps y guarda en leads.json
3. Importar leads.json en el CRM (crm/index.html)
4. Filtrar los que tienen email y estado "nuevo"
PASO 2 · Enviar Emails Masivos
1. Correr: python outreach.py --limite 30
2. El script envía emails personalizados por tipo de negocio (nail salon, dental, etc.)
3. Se actualiza leads.json marcando estado "contactado"
4. Se guarda outreach_log.json con el registro de cada envío
5. Ver resultados en CRM → Tab "Historial de Envíos"
PASO 3 · Cliente Responde Interesado
1. El cliente responde al email o llama al demo: +1 (786) 605-6242
2. En el CRM marcar lead como "Interesado"
3. Ofrecer prueba demo gratuita de 7 días
4. Ir a Nueva Licencia → generar código DEMO para ese cliente
5. Enviarle: panel-cliente.html + su código de licencia
PASO 4 · Cerrar la Venta
1. El cliente prueba el panel 7 días (demo)
2. Hacer follow-up: "¿Cómo está funcionando?"
3. Ofrecer plan Básico $99/mes → cobrar por Zelle/CashApp/Venmo
4. Una vez pagado → generar nueva licencia BASE en este panel
5. Enviarle el nuevo código y marcar en CRM como "Cerrado ✅"
